Le projet "MedievAIl BAIttle GenerAIl" est un simulateur de batailles médiévales en 2.5D développé en Python.

--> L'objectif principal est de programmer des Intelligences Artificielles (IA) pour commander des armées et s'affronter sur un champ de bataille et d'ensuite pourvoir réaliser un tournoi pour déclarer un ChAImpion.

Ce projet est réalisé par une équipe de 7 étudiants -INSA CVL

## Comment Installer et Lancer le Projet

Pour mettre en place l'environnement de développement et lancer le simulateur:

**1. Cloner le Dépôt**

Pour toute modification du code, on utilise...
```bash
git clone https://github.com/ItzzNolan/Project_Python
```
...Ou pour toute version alternative juste du main :
```bash
git clone --branch projet_python_main --single-branch --depth 1 https://github.com/ItzzNolan/Project_Python.git
```
