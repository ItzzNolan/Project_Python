Le projet "MedievAIl BAIttle GenerAIl" est un simulateur de batailles médiévales en 2.5D développé en Python.

--> L'objectif principal est de programmer des Intelligences Artificielles (IA) pour commander des armées et s'affronter sur un champ de bataille et d'ensuite pourvoir réaliser un tournoi pour déclarer un ChAImpion.
